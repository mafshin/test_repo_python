# Problem 018
Write a program which prints all Twin-Primes less than input number.

# Persian Description
برنامه ای بنویسید که یک عدد از ورودی دریافت کرده و تمام جفت اعداد اول دوقلوی (Twin Primes) کوچکتر از آن را چاپ کند

# Sample Input/Output

## Input:
```
20
```

## Output: 
```
3, 5
5, 7,
11,13
17,19
```
